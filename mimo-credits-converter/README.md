# MiMo Credits → 人民币

将小米 MiMo 开放平台的 Credits 自动换算为人民币，并展示今日/月度 Token 消耗统计。

## 功能

- **Credits → 元**：自动将套餐页面的 Credits 数值换算为人民币显示
- **点击切换**：点击已转换的数值可在 Credits ↔ 人民币之间切换
- **今日消耗**：展示当日 Token 消耗，按模型分类
- **月度消耗**：展示计费周期内 Token 消耗，含全部合计
- **模型区分**：`mimo-v2.5` 和 `mimo-v2.5-pro` 分开统计，各自费率独立计算

## 安装

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择本项目目录

## 费率参考

| 模型 | 输入（命中缓存）| 输入（未命中缓存）| 输出 |
|------|---------------|-----------------|------|
| mimo-v2.5 | ¥0.02/M | ¥1.00/M | ¥2.00/M |
| mimo-v2.5-pro | ¥0.025/M | ¥3.00/M | ¥6.00/M |
